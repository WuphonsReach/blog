--- layout: default title: 'FO3: Using the console to raise Deputy Weld from the dead' date: '2010-08-30T20:57:00.000-04:00' author: Wuphon's Reach tags: - Fallout3 modified_time: '2010-08-31T08:59:38.729-04:00' blogger_id: tag:blogger.com,1999:blog-3097974.post-3079911771445965250 --- An example of using the console to resurrect a dead NPC. This particular example is what to do if you can find the dead body. Due to the mods that I'm using (Mart's Mutant Mod w/ Increased Spawns), a lot of the exterior NPCs run the risk of early death at the hands (or claws) of wasteland enemies. This also happens in the base Fallout 3 game (usually with the roaming trade caravans or Deputy Weld).

1. Get near the body, and look at it
2. Open the console (the tilde key on the PC)
3. Click on the NPC's body, the ID will show at the top of the screen
4. "prid ID" will select that object
5. kill, resurrect, disable, enable
6. Exit the console

If the NPC was killed far away from where they should be, you should use the Player.PlaceAtMe command instead after the "prid" line.
